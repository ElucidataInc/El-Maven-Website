About

ElMaven is an open source mass spectrometry data processing engine that is optimal for isotopomer labeling and global metabolomic profiling experiments. It's designed to reduce complexity in metabolomics analysis by using an intuitive interface for exploring the experimental data. The tool aims to make research faster and easier than ever before.

History

The birth of El Maven can be traced back to another open source project, Maven, an open source LC-MS data processing engine developed by Euguene Melamud. We decided to use El Maven as the working title for the project for few reasons. One, we wanted to acknowledge the original project. We also want to commit ourselves to create “the” version of Maven that is widely used and supported by the community. Second, El bears the stamp of Elucidata.
